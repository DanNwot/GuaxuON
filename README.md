# 🗑️ Sistema de Informação sobre Coleta de Lixo

Projeto desenvolvido com o objetivo de melhorar o acesso da população de Guaxupé às informações sobre horários e funcionamento da coleta de lixo, utilizando tecnologias modernas e boas práticas de desenvolvimento de software.

---

## 🎯 Objetivo

Desenvolver uma solução tecnológica eficiente que resolva ou minimize problemas relacionados à falta de informação sobre a coleta de lixo.

A proposta busca facilitar o acesso da população guaxupeana a dados importantes, como dias e horários de coleta, por meio de um sistema simples, intuitivo e com alto nível de abstração para o usuário final.

---

## 🚨 Problemas Identificados

* Falta de informação clara sobre horários de coleta de lixo
* Dificuldade de acesso a informações atualizadas
* Baixa organização no descarte de resíduos
* Impactos ambientais causados pelo descarte incorreto

---

## 💡 Solução Proposta

Desenvolvimento de um sistema web que permitirá:

* Consulta de horários de coleta por bairro
* Interface simples e acessível
* Organização das informações de forma clara
* Possível expansão para notificações futuras

---

## 🛠️ Tecnologias Utilizadas

* Python
* Django

* Engenharia de Software

---

## ⚙️ Funcionalidades

* 📍 Consulta de coleta por localização
* 🕒 Visualização de dias e horários
* 📱 Interface responsiva (desktop e celular)
* 📊 Organização e centralização das informações

---

## 🧪 Metodologia

O projeto segue princípios de Engenharia de Software, incluindo:

* Levantamento de requisitos
* Modelagem do sistema
* Desenvolvimento orientado a objetos
* Testes e validação

---

## 📈 Resultados Esperados

* Maior conscientização da população
* Redução de lixo descartado incorretamente
* Facilidade no acesso às informações
* Melhoria na organização urbana
